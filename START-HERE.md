## ⚡ Super Quick Start

```bash
# 1. Install dependencies
./setup.sh

# 2. Start backend (new terminal)
cd backend && npm run dev

# 3. Start frontend (new terminal)
cd frontend && npm run dev

# 4. Open browser
# Visit: http://localhost:3000
```
